### Removed
- Removed Oblique view mode. The orthogonal mode viewports can be rotated freely, making oblique mode redundant.

### Changed
- Renamed the "arbitrary" mode concept to "flight mode" throughout the codebase. The `clippingDistanceArbitrary` user setting has been renamed to `clippingDistanceFlight`.
