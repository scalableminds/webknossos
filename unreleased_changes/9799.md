### Fixed
- Fixed several WebGL/three.js memory leaks (render targets during node picking and screenshots, bounding box geometries, segment meshes, skeleton shader resources, quick-select texture masks and plane geometries are now properly disposed) and reduced GPU upload overhead (partial buffer/texture updates for skeleton edits, proofreading hover highlighting and tree colors). Also, the GPU memory of an annotation is now released deterministically when it is closed, and idle tabs no longer run permanent per-layer render loops.
