Api

webKnossos Public Frontend API.

Api
Version: 1
Author: scalabe minds
Properties
tracing (TracingApi) : All methods related to getting tracings.
data (DataApi) : All methods related to getting binary data / layers.
user (UserApi) : All methods related to getting / setting the user's personal tracing configuration.
utils (UtilsApi) : Utitility methods for controlling wK.
Example
window.webknossos.apiReady(1, (api) => {
    const nodes = api.tracing.getAllNodes();
    const dataLayerNames = api.data.getLayerNames();
    const userConfiguration = api.user.getConfiguration();
    const keyHandler = api.utils.registerKeyHandler("enter", () => console.log("Welcome"));
 });
