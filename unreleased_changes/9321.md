### Added
- Added the option to disable inserting the organization directory for the paths selected during the uploadToPaths protocol: New config option `webKnossos.datasets.uploadToPaths.insertOrganizationDirectory` (default true)

### Migration
- Renamed config keys: `webKnossos.datasets.uploadToPathsPrefixes` to `webKnossos.datasets.uploadToPaths.prefixes`, `webKnossos.datasets.uploadToPathsInfix` to `webKnossos.datasets.uploadToPaths.infix`.
